## Introduction

Rand Portfolio is a personal portfolio website designed to showcase your projects, skills, and experiences to potential employers, clients, or collaborators. It provides a platform to highlight your expertise, display your work, and establish your online presence.

## Features

- **Project Showcase:** Display your projects with descriptions, screenshots, and links.
- **Skills Section:** Highlight your technical skills, tools, and technologies.
- **About Me:** Introduce yourself, your background, and your interests.
- **Contact Form:** Allow visitors to get in touch with you via a contact form.
- **Responsive Design:** Ensure a seamless experience across devices and screen sizes.

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript, React.js
- **Styling:** CSS Modules, SCSS
- **Routing:** React Router
- **Form Handling:** React Hook Form
- **Deployment:** GitHub Pages

## Installation

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/RandAttrah8088/rand-portfolio.git
   ```

2. Navigate to the project directory:

   ```bash
   cd rand-portfolio
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

## Usage

Once you've installed the dependencies, you can start the development server:

```bash
npm start
```

This will launch the portfolio website in your default web browser. You can then view and interact with the website locally.
